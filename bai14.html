BÀI 14: ĐỊNH DẠNG VĂN BẢN BẰNG CSS

IV. Luyện tập

1) Mỗi phông chữ sau đây thuộc loại nào:

a. Times New Roman

b. Courier New

c. Abadi

d. Bradley Hand ITC

e. Berlin Sans FB

f. Algerian

g. Consolas

h. Cascadia

2) Hãy liệt kê các thuộc tính CSS liên quan đến định dạng đoạn văn bản sau:

Trời Hà Nội xanh

Sáng tác: Văn Ký

Xanh xanh thắm bầu trời xanh Hà Nội

Hồ Gươm xanh như mái tóc em xanh

- Căn đoạn văn bản

- Màu chữ

- Kiểu chữ

V. Vận dụng

1) Tìm hiểu thêm các thuộc tính phông chữ:

- font-variant: Thuộc tính này được sử dụng để xác định kiểu chữ viết hoa của văn bản.

- letter-spacing: Thuộc tính này xác định khoảng cách giữa các kí tự trong một đoạn văn bản.

- word-spacing: Thuộc tính này xác định khoảng cách giữa các từ trong một đoạn văn bản.

- text-shadow: Thuộc tính này cho phép bạn thêm hiệu ứng bóng cho văn bản.

2) Thiết lập tệp CSS:

Kiểu 1:

/* stylel.css */

body {

font-family: Arial, sans-serif;

font-size: 16px;

color: #333;

}

h1 {

font-family: "Times New Roman", serif;

font-size: 28px;

color: #990000;

}

p {

font-family: "Courier New", monospace;

font-size: 14px;

color: #666666;

}

Kiểu 2:

/* style2.css */

body {

font-family: Verdana, Arial, sans-serif;

font-size: 18px;

color: #555555;

}

h1 {

font-family: "Helvetica Neue", Arial, sans-serif;

font-size: 32px;

color: #006600;

}

p {

font-family: Georgia, serif;

font-size: 16px;

color: #888888;

}
